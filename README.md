# Pokedex API

a RESTful API that holds data and stats about Pokémon

## Tools Used

* Node
    - runtime environment that allows JavaScript to be used as a server-side language

* Express
    - framework for creating web applications and API's with Node

* Nodemon
    - development tool that automatically restarts your Node application when there have been changes to a file. (no need to stop and re-run application to see reflected changes)

* Postman
    - platform for testing API functionality (calls and responses)

* ChatGPT
    - one of the most powerful tools for developers
